# 로또 번호 추첨기

## 기능

### 로또 번호 추첨
> 1 ~ 45 의 숫자를 랜덤으로 중복없이 6개를 뽑고 오름차순으로 정렬

### 720 연금 복권 추첨
> 1 ~ 5조, 000000 ~ 999999 의 숫자를 랜덤으로 추출
